GITHUB

1. КОГДА НАЧИНАЕШЬ РАБОТАТЬ
cd [путь до папки]
git pull

2. ПОСЛЕ ОКОНЧАНИЯ РАБОТЫ
git status (убедиться что комп видит изменения. все изменения красные)
git add -A (добавляешь изменения в специальную скрытую папку)
git status (убедиться что изменения произошли. все изменения зелёные)
git commit -m "comment" (комментируешь изменения)
git push (заливаешь в гит)
